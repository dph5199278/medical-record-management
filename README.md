# medical-record-management
medical record management(病历管理系统)

本系统是在网上得到的一个SSM系统改造而来，可以用来做毕设，这个项目商业公司也看不上，个人玩玩。

## 技术栈
### 后台
* Jdk1.8
* Spring Boot 2.6.2
* Mybatis
* MySql8.0以上
* JSP

### 前端
* jQuery
* jQuery EasyUI 1.3.3
* layui-v2.0.2
* Bootstrap v2.3.0
